Full Stack Web Developer

Full-Stack Web Developer leveraging a background in problem solving to create a more immersive user experience on the web. Recently earned a certificate in Full Stack Development from the University of Denver, with newly developed skills in JavaScript, CSS, React.js, and responsive web design. Known as an innovative critical-thinker with a passion for developing apps, with a focus on mobile-first design and development. High attention to detail goes into each project built with the goal of providing the user a superior experience. Recently worked with a team to develop a single-page app that helped the user search and retrieve relative news articles online. Excited to leverage and learn new skills in the future as part of a fast-paced, quality-driven team in order to build better experiences on the web.
